# Tech Stack

## Frontend Framework
- **React**: UI library for building component-based interfaces.
- **TypeScript**: Static typing for improved developer experience and code safety.
- **Vite**: Next-generation frontend tooling for fast development and build times.

## Styling
- **CSS Modules**: Scoped CSS for components to avoid style leakage.
- **PostCSS**: Processing CSS with plugins (if applicable).

## Routing
- **React Router DOM**: Client-side routing for seamless navigation.

## Forms & Integrations
- **Formspree**: Backend-as-a-service for handling contact form submissions.

## Testing & Quality
- **Vitest**: Unit testing framework compatible with Vite.
- **ESLint**: Linter for identifying and reporting on patterns in JavaScript/TypeScript.
- **Prettier**: Code formater for consistent style.

## Deployment
- **Vercel**: Platform for static and serverless deployment.
    - Continuous Integration via Git.
    - Environment Variable management for secrets.
